# koishi-plugin-starrail-gachalog

《崩坏 星穹铁道》抽卡分析


## 目录结构
### 已经做好了的
- [x] Url导入抽卡记录
- [x] 抽卡分析
- [x] 图像渲染（简陋版）

### 使用方法
- 绑定url
    - sr.抽卡链接 + 抽卡历史的链接
- sr.更新抽卡
- sr.抽卡分析
  - -t 卡池代码
